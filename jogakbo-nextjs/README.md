# 조각보 frontend

클레이튼 기반 기부 플랫폼

## 개발자(진행중)

- 김다운: BE(Server, DB)
- 백시아: FE
- 배지선: BE(Smart Contract)

## 기술

`Next.js` `Typescript` `React.js` `Recoil` `Styled-components` `caver-js`

## 특징

- next.js / typescript 마이그레이션
- 린트/프리티어 설정
- 캠페인 페이지 데이터 SSR 구현
- SSR 활용한 파라미터 데이터 전달
- 토큰 생성 및 쿠키를 활용한 로그인 유지
- caver.js를 활용한 데이터 교환
- 스마트 컨트랙트 함수 비동기 처리
- klip API를 활용한 카카오 클립 지갑 연동
- Recoil을 활용한 상태관리
- 캠페인 상태별 필터링 기능 구현
- 기부율 데이터 시각화
- dynamic routing을 활용한 캠페인 상세 페이지 구현
- chrome, safari, firefox, opera, edge에 대한 크로스브라우징 테스트
- 미디어쿼리를 이용한 반응형 구현
- 캠페인 카테고리 추가 및 카테고리별 필터링 기능 구현
- 다크모드 기능 추가
- 카이카스 지갑 로그인 연동 추가
- DB 관리를 위한 관리자 페이지 추가

## 스크린샷

- 보안상 관리자 페이지는 올리지 않습니다
